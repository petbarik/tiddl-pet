## Installation

#Mac/Linux
```bash
git clone https://github.com/petbarik/tiddl-pet.git && cd tiddl-pet && python3 -m venv .venv && source .venv/bin/activate && pip install --upgrade pip setuptools wheel && pip install .
```
#Windows
```bash
git clone https://github.com/petbarik/tiddl-pet.git; cd tiddl-pet; python -m venv .venv; .\.venv\Scripts\Activate.ps1; pip install --upgrade pip setuptools wheel; pip install .
```
